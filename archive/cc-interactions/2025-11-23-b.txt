       40 +  - Post-season corrected versions (vs field versions)
       41 +

       43 +  - Uncurated scans (when Finalized version exists)
       44 +  - Less complete diaries (smaller file size)
       45 +  - Translations (when BG original exists)
       46 +  - Field versions (when corrected version exists)
       47 +  
       48 +  **SUPPLEMENTAL:** Additional perspective, not alternative

       49 +  - Personal/individual diaries (vs official team diaries)
       50 +  - Project leader expedition journals (overall perspective)
       51 +  - Daily journals (when compiled journal exists)
